# md

minimal markdown web blog

Markdown specification:

# Headers

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

Looks like :
# H1
## H2
### H3
#### H4
##### H5
###### H6

# Lists

```
* Item 1
* Item 2
* Item 3
```

Looks like:

* Item 1
* Item 2
* Item 3


# Links 

```
[name of link](link)

[search](google.com)
```

Looks like:

[name of link](link)

[search](google.com)

# Images

```
![logo](/static/logo.png)
```

Looks like:

![logo](/static/logo.png)


# Block of text

```
Block begin from ``` and ended ```.
```

# Tables

```

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

```

Looks like:

| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
